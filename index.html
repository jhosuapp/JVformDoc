Atributo para mensajes de error

data-error - Es un atributo personalizado, el cual nos permite personalizar el mensaje de error

                        
                            <
                            input 
                            data-error
                            
                            =
                            "Mensaje de ejemplo"
                            >
                        
                        

                            
copiado

En caso de que no pongas este atributo, se tomara un mensaje por defecto, el cual toma el texto "ingresa" + name del campo

Atributo para campos no obligatorios

data-mandatory - Es un atributo personalizado, el cual permite que un campo no sea obligatorio

                        
                            <
                            input 
                            data-mandatory
                            
                            =
                            "false"
                            >
                        
                        

                            
copiado

Por defecto todos los campos son obligatorios, no es necesario que añadas este atributo a menos que necesites tener campos no obligatorios

Atributos para validaciones que vienen por defecto

data-validation - Es un atributo que nos permite añadir validaciones por medio de unas expresiones regulares predefinidas, los parametros que puede recibir son los siguientes:

  • data-validation="text" - nos añade la siguiente expresión regular como validación "/^[a-zA-ZÀ-ÿ\s]{3,40}$/"
  • data-validation="email" - nos añade la siguiente expresión regular como validación "/^[a-zA-ZÀ-ÿ\s]{3,40}$/"
  • data-validation="phone" - nos añade la siguiente expresión regular como validación "/^[a-zA-ZÀ-ÿ\s]{3,40}$/"
  • data-validation="password" - nos añade la siguiente expresión regular como validación "/^[a-zA-ZÀ-ÿ\s]{3,40}$/"
  • data-validation="message" - nos añade la siguiente expresión regular como validación "/^[a-zA-ZÀ-ÿ\s]{3,40}$/"
  • data-validation="url" - nos añade la siguiente expresión regular como validación "/^[a-zA-ZÀ-ÿ\s]{3,40}$/"
                        
                            <
                            input 
                            data-validation
                            
                            =
                            "text"
                            >
                        
                        

                            
copiado

Atributos para validaciones por medio de expresiones regulares dinamicas

data-expresion - Es un atributo que nos permite añadir validaciones por medio de expresiones regulares definidas por nosotros:

                        
                            <
                            input 
                            data-expresion
                            
                            =
                            "^(\+)?[\d\s\-\.]{5,30}$"
                            >
                        
                        

                            
copiado

Esta expresión de ejemplo nos permite añadir solo números, cierta cantidad de caracteres especiales y un mínimo de 5 y un máximo de 30 caracteres

Atributo para extensiones permitidas

data-fileallowed - Es un atributo personalizado, el cual nos permite limitar el tipo de archivos que suben

                        
                            <
                            input 
                            type
                            =
                            "file" 
                            data-fileallowed
                            
                            =
                            "jpg,pdf,png"
                            >
                        
                        

                            
copiado

Separar las extensiones con "," y sin espacios para su correcta funcionalidad

Atributo para peso máximo permitido

data-filesize - Es un atributo personalizado, el cual nos permite limitar el tamaño de los archivos que suben

                        
                            <
                            input 
                            type
                            =
                            "file" 
                            data-filesize
                            
                            =
                            "1000000"
                            >
                        
                        

                            
copiado

El peso se calcula tomando como base los bytes, por lo tanto, si añadimos el siguiente número 1000000, el límite de peso sería de un 1MB

Atributos para confirmación de contraseña

data-confirmpass - Es un atributo personalizado, el cual nos permite añadir un segundo campo de contraseña para la confirmación, con el true, indicamos que es un campo para confirmar contraseña, por lo tanto, tendrá que hacer match con otro campo de contraseña

                        
                            <
                            input 
                            type
                            =
                            "password" 
                            data-confirmpass
                            
                            =
                            "true"
                            >
                        
                        

                            
copiado

Sí necesita la confirmación de contraseña, debe añadir el atributo que se explica a continuación para su correcto funcionamiento

                        
                            <
                            input 
                            type
                            =
                            "password" 
                            data-confirmpass
                            
                            =
                            "true" 
                            data-match
                            
                            =
                            "#id del campo"
                            >
                        
                        

                            
copiado

En este atributo debe añadir el id del campo con el que debe hacer match
